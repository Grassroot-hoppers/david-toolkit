# Demo Script

## 1. Open

Open the dashboard and start at the header.

Message:

> This does not replace the POS. It turns the shop’s ugly exports into a decision surface.

## 2. Show context

Point to:

- weather
- holiday or calendar pressure
- macro health strip

Message:

> The dashboard opens with context, not just numbers. The shop lives in weather, rhythm, and timing.

## 3. Show supplier command panels

Walk one supplier card.

Message:

> The operator should see order / watch / skip, plus the evidence behind each signal.

## 4. Show interpreted cards

Point to raw vs interpreted evidence.

Message:

> This is the critical distinction: we do not pretend to know. We make inference visible and score its confidence.

## 5. Show workflow map alongside it

Message:

> The dashboard tells you what the shop is saying. The workflow map tells you how the shop should run.

## 6. Close

Message:

> This is my first serious open-source software step: old POS in, clean intelligence out, and the repo is ready for strangers.
